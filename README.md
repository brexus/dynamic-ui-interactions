
# Dynamic UI Interactions Assignment

This is the Dynamic UI Interactions assignment for The Odin Project course.
While creating this site, I practiced creating an interactive, responsive navigation bar and created a photo slider.
I also learned how to use ESLint.

## Demo live

https://brexus.github.io/dynamic-ui-interactions


## Tech Stack

HTML, CSS, JS, WebPack, ESLint
